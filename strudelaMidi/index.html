🎹 Cifrados a MIDI

Estructura recursiva inspirada en Strudel. Soporta anidamiento infinito y división de tiempo cíclica.

Manual rápido:
• [a, b, c] -> Acorde (suenan juntas)
• [a] [b] -> Sucesión (subdivide el tiempo disponible)
• /n -> Multiplica la duración del bloque por n
• Soporta #, s (sostenido) y b (bemol).
